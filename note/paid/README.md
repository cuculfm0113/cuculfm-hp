# note（有料）コンテンツ管理フォルダ

このフォルダは、**noteで販売する有料記事の原稿（下書き）**を管理するためのものです。

## 目的
- シリーズごとに原稿を整理し、更新・差し替え・販売単位（単品/まとめ）を管理しやすくする
- noteへ貼り付ける前提で、Markdownで本文を管理する（見出し・箇条書き・表を安定させる）

## ルール（推奨）
- **1ファイル=1記事**（noteへそのまま貼り付け）
- ファイル名は `NN-英数字スラッグ.md`（例: `01-chatgpt-basics.md`）
- 冒頭に「販売向けメタ情報（下書き用）」のブロックを置く（※note公開時は不要なら削除）
- 画像が必要な場合は、note側で差し込む前提で `TODO: 画像` を本文に残す

## 構成
- `index.md`: 有料記事の全体目次（シリーズ一覧）
- `_template.md`: 記事テンプレ（コピペして新規作成）
- `chatgpt/`: ChatGPTシリーズ
- `claude/`: Claudeシリーズ
- `cursor/`: Cursorシリーズ
- `research-tools/`: リサーチ系ツール（Perplexity / NotebookLM等）シリーズ
